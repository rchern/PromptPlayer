  <action>
    Create `src/renderer/src/types/presentation.ts` with these interfaces:

    ```typescript
    export interface SessionRef {
      sessionId: string           // References StoredSession.sessionId
      displayName: string         // User-editable friendly name
      sortKey: string             // ISO timestamp for chronological ordering (from SessionMetadata.firstTimestamp)
    }

    export interface PresentationSection {
      id: string                  // crypto.randomUUID()
      name: string                // User-editable section name
      sessionRefs: SessionRef[]   // Ordered by sortKey (chronological)
    }

    export interface Presentation {
      id: string                  // crypto.randomUUID()
      name: string                // User-editable presentation name
      sections: PresentationSection[]
      createdAt: number           // Date.now()
      updatedAt: number           // Date.now()
    }
    ```

    Create `src/renderer/src/utils/presentationUtils.ts` with these exported functions:

    1. `generateSessionDisplayName(meta: SessionMetadata): string` - Derives friendly name from firstUserMessage metadata:
       - GSD commands (`/gsd:plan-phase 3`) -> "Plan Phase 3" (split kebab on `-`, title-case each word, append args)
       - Other slash commands (`/command args`) -> "Command: args"
       - Plain messages -> truncated to 60 chars with "..." suffix
       - Fallback (no firstUserMessage) -> "Session {first 8 chars of sessionId}"

    2. `generatePresentationName(sessions: SessionMetadata[]): string` - Auto-names presentation:
       - Single project: "{projectFolder} - {date}" where date is "Feb 22, 2026" format via toLocaleDateString('en-US', { month: 'short', day: 'numeric', year: 'numeric' })
       - Mixed projects: "Presentation - {date}"

    3. `sortSessionRefsChronologically(refs: SessionRef[]): SessionRef[]` - Sorts by sortKey (ISO string). Missing sortKeys sort last. Returns new array (no mutation).

    4. `createPresentationFromSessions(sessions: SessionMetadata[]): Presentation` - Creates a full Presentation object:
       - One PresentationSection per session, section name = generateSessionDisplayName(session)
       - Each section has one SessionRef with sessionId, displayName (same as section name), sortKey = session.firstTimestamp ?? ''
       - Sections sorted chronologically by their session's sortKey
       - Presentation name from generatePresentationName
       - IDs from crypto.randomUUID()
       - createdAt and updatedAt from Date.now()

    Import SessionMetadata from '../types/pipeline' and Presentation types from '../types/presentation'.
  </action>

  </verify>
