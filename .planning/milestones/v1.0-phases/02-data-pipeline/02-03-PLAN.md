  <action>
Create `src/renderer/src/stores/sessionStore.ts` using Zustand (same pattern as existing `appStore.ts`).

Import types from `../types/pipeline`.

**State interface `SessionState`:**
```typescript
interface SessionState {
  // Discovery
  discoveredSessions: SessionMetadata[]
  isDiscovering: boolean
  discoveryError: string | null

  // Active session (full parse)
  activeSession: StitchedSession | null
  activeSessionId: string | null
  isParsing: boolean
  parseError: string | null

  // Stored sessions (for presentations)
  storedSessions: StoredSession[]

  // Actions
  discover: (baseDir?: string) => Promise<void>
  browseAndDiscover: () => Promise<void>
  parseSession: (filePath: string, sessionId: string) => Promise<void>
  loadStoredSessions: () => Promise<void>
  saveSessionToStorage: (session: StoredSession) => Promise<void>
  removeSessionFromStorage: (sessionId: string) => Promise<void>
  clearActiveSession: () => void
}
```

**Action implementations:**

1. `discover(baseDir?)`:
   - Set `isDiscovering: true`, `discoveryError: null`
   - Call `window.electronAPI.discoverSessions(baseDir)`
   - On success: set `discoveredSessions` to result, `isDiscovering: false`
   - On error: set `discoveryError` to error message, `isDiscovering: false`

2. `browseAndDiscover()`:
   - Call `window.electronAPI.browseDirectory()`
   - If result is null (user cancelled), return early
   - Otherwise call `discover(result)` with the selected path

3. `parseSession(filePath, sessionId)`:
   - Set `isParsing: true`, `parseError: null`, `activeSessionId: sessionId`
   - Call `window.electronAPI.parseSession(filePath)`
   - On success: set `activeSession` to result, `isParsing: false`
   - On error: set `parseError` to error message, `isParsing: false`, `activeSession: null`

4. `loadStoredSessions()`:
   - Call `window.electronAPI.getStoredSessions()`
   - Set `storedSessions` to result

5. `saveSessionToStorage(session)`:
   - Call `window.electronAPI.saveStoredSession(session)`
   - Refresh by calling `loadStoredSessions()`

6. `removeSessionFromStorage(sessionId)`:
   - Call `window.electronAPI.removeStoredSession(sessionId)`
   - Refresh by calling `loadStoredSessions()`

7. `clearActiveSession()`:
   - Set `activeSession: null`, `activeSessionId: null`, `parseError: null`

Export `useSessionStore` as the Zustand hook.
