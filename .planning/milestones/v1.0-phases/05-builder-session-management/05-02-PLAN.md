       CORRECTION: Add a new IPC handler in this task. Actually -- let's handle this differently. The `importDroppedFiles` action should call a new IPC `pipeline:importDroppedFiles` that takes an array of file paths and returns SessionMetadata[].

       WAIT -- let me reconsider. Plan 01 already added the main process `extractSessionMetadata` function as exported. We need an IPC that takes file paths and returns metadata. Let me add this IPC to the main process as part of this task.

       Actually, the cleaner approach: Add `pipeline:importFromPaths` IPC handler to `src/main/index.ts` that accepts `string[]` (file paths), runs `extractSessionMetadata` on each, and returns `SessionMetadata[]`. Also update preload and types.

       In `src/main/index.ts`, add:
       ```typescript
       ipcMain.handle('pipeline:importFromPaths', async (_event, filePaths: string[]) => {
         const sessions: SessionMetadata[] = []
         for (const fp of filePaths) {
           const projectFolder = basename(dirname(fp))
           const meta = await extractSessionMetadata(fp, projectFolder)
           sessions.push(meta)
         }
         return sessions
       })
       ```
       Import `basename` and `dirname` from 'path' (basename is already imported, add dirname).

       In `src/preload/index.ts`, add:
       ```typescript
       importFromPaths: (filePaths: string[]): Promise<unknown> =>
         ipcRenderer.invoke('pipeline:importFromPaths', filePaths),
       ```

       In `src/renderer/src/types/electron.d.ts`, add to ElectronAPI:
       ```typescript
       importFromPaths: (filePaths: string[]) => Promise<SessionMetadata[]>
       ```

       Then `importDroppedFiles` in the store:
       ```typescript
       importDroppedFiles: async (filePaths: string[]): Promise<void> => {
         if (filePaths.length === 0) return
         set({ isImporting: true })
         try {
           const newSessions = await window.electronAPI.importFromPaths(filePaths)
           const existing = get().discoveredSessions
           const existingIds = new Set(existing.map(s => s.sessionId))
           const unique = newSessions.filter(s => !existingIds.has(s.sessionId))
           set({
             discoveredSessions: [...existing, ...unique],
             importCount: unique.length,
             isImporting: false
           })
         } catch {
           set({ isImporting: false })
         }
       }
       ```

       - Store the matching sessionIds in a new state field `deepSearchMatchIds: Set<string>` (add to state interface, default `new Set()`)

       Add `deepSearchMatchIds: string[]` to state (use array for Zustand serialization, convert to Set in consumer). Default `[]`. Add `isDeepSearching: boolean` default `false`.
