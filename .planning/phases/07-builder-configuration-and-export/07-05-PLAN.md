  <name>Task 1: Parse and save sessions to StoredSession storage during presentation creation and session addition</name>
  <files>src/renderer/src/stores/presentationStore.ts</files>
  <action>
Modify `createPresentation` and `addSessions` in presentationStore.ts to parse each session's JSONL file and save it as a StoredSession via the existing IPC bridge.

**In `createPresentation`:**
1. After creating the presentation object, iterate over the `sessions: SessionMetadata[]` array
2. For each session, call `await window.electronAPI.parseSession(session.filePath)` to get a `StitchedSession` with parsed messages
3. Build a `StoredSession` object: `{ sessionId: session.sessionId, projectFolder: session.projectFolder, originalFilePath: session.filePath, messages: stitchedSession.messages, metadata: session, addedAt: Date.now() }`
4. Call `await window.electronAPI.saveStoredSession(storedSession)` to persist it
5. The `createPresentation` method is already async, so this fits naturally

**In `addSessions`:**
1. This method is currently synchronous (returns void). Change it to `async` (returns `Promise<void>`)
2. Before the `persistPresentation(updated)` call, iterate over the `sessions: SessionMetadata[]` array
3. For each session, parse and save to StoredSession storage using the same pattern as createPresentation above
4. Import `StitchedSession` type from `'../types/pipeline'` (add to existing import)

**Critical implementation detail:**
- `parseSession` returns `StitchedSession` (which has `.messages: ParsedMessage[]`), NOT `StoredSession`
- You must construct the `StoredSession` from the `StitchedSession.messages` + the `SessionMetadata`
- Wrap each parseSession call in try/catch to handle file-not-found gracefully (log warning but don't fail the entire creation)
- The addSessions signature in the PresentationState interface must also be updated from `(sessions: SessionMetadata[]) => void` to `(sessions: SessionMetadata[]) => Promise<void>`

   ```
   // Warn if any referenced sessions are missing from storage
   const foundIds = new Set(sessions.map((s) => s.sessionId))
   const missingIds = [...referencedIds].filter((id) => !foundIds.has(id))
   if (missingIds.length > 0) {
     console.warn(`Export warning: ${missingIds.length} session(s) not found in storage: ${missingIds.join(', ')}`)
   }
   ```
