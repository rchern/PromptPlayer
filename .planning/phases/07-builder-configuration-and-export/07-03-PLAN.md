  <action>
Add three new IPC handlers to `src/main/index.ts` inside the `createWindow` function, in a new section `// ---------- IPC: Presentation Export/Import ----------`:

1. **`presentation:export`** handler:
   - Receives: `presentationId: string` from renderer
   - Logic:
     a. Read all presentations from `getPresentations()`, find the one matching `presentationId`
     b. If not found, throw/return error
     c. Collect all session IDs referenced in the presentation (flatten `sections[].sessionRefs[].sessionId`)
     d. Read all stored sessions from `getStoredSessions()`, filter to only those referenced
     e. Assemble `PromptPlayFile` object: `{ presentation, sessions }`
     f. Show save dialog:
        ```typescript
        const result = await dialog.showSaveDialog(mainWindow, {
          title: 'Export Presentation',
          defaultPath: `${presentation.name.replace(/[<>:"/\\|?*]/g, '_')}.promptplay`,
          filters: [
            { name: 'PromptPlay Presentations', extensions: ['promptplay'] },
            { name: 'All Files', extensions: ['*'] }
          ]
        })
        ```
     g. If not canceled, write JSON: `writeFileSync(result.filePath, JSON.stringify(exportData, null, 2), 'utf-8')`
     h. Return `{ filePath: result.filePath, canceled: false }` on success, or `{ canceled: true }` if canceled
   - Import `PromptPlayFile` type from `./pipeline/types` (or define locally if circular issues)

2. **`presentation:import`** handler:
   - Receives: no arguments
   - Logic:
     a. Show open dialog:
        ```typescript
        const result = await dialog.showOpenDialog(mainWindow, {
          title: 'Open Presentation',
          filters: [
            { name: 'PromptPlay Presentations', extensions: ['promptplay'] },
            { name: 'All Files', extensions: ['*'] }
          ],
          properties: ['openFile']
        })
        ```
     b. If canceled or no files, return `null`
     c. Read the file: `readFileSync(result.filePaths[0], 'utf-8')`
     d. Parse JSON: `JSON.parse(data) as PromptPlayFile`
     e. Basic validation: check that `parsed.presentation` exists and `parsed.sessions` is an array
     f. Return `{ ...parsed, filePath: result.filePaths[0] }` (include filePath so renderer knows the source)

3. **`presentation:saveToPath`** handler:
   - Receives: `presentationId: string, filePath: string`
   - Logic:
     a. Same data assembly as export (read presentation + sessions from stores)
     b. Write directly to `filePath` without showing a dialog (this is the Ctrl+S overwrite path)
     c. Return `{ success: true }`
   - Per Pitfall 4: This is only called when `sourceFilePath` is set on the presentation (renderer enforces this)

IMPORTANT: All file I/O uses synchronous `writeFileSync`/`readFileSync` -- consistent with existing patterns. File sizes are 2-15 MB, well within sync I/O tolerance.

NOTE: The `PromptPlayFile` type was defined in Plan 01 in `src/main/pipeline/types.ts`. Import it from there.
