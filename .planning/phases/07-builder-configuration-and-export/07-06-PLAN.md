**Goal:** Clicking a session entry in the assembly outline's PresentationOutline triggers parseSession for that session, populating the live preview panel below.

**In PresentationOutline.tsx:**
1. Add a new prop to the component: `onSessionClick?: (sessionId: string) => void`
2. Pass this callback to each `<SessionEntry>` component. SessionEntry currently has `onRename` and `onRemove` props.
3. Add an `onClick` prop to SessionEntry. In SessionEntry, make the session name/row clickable (e.g., wrap in a div with onClick that calls the callback).

Alternatively, if modifying SessionEntry is complex, add a simple click handler directly in PresentationOutline's map of sessionRefs. Wrap each `<SessionEntry>` in a div with `onClick={() => onSessionClick?.(ref.sessionId)}` and add `cursor: pointer` styling.

**In Builder.tsx:**
1. The assembly view already has a session preview panel at the bottom (lines 420-483) that renders when `activeSession` is not null.
2. To make this work, we need to know the session's filePath from its sessionId. The `discoveredSessions` array has metadata keyed by sessionId. But sessions from imported .promptplay files may not be in discoveredSessions.
3. Create a lookup: when a session in the outline is clicked, find its filePath from either:
   - `discoveredSessions` metadata (for sessions discovered from JSONL files), OR
   - `storedSessions` from the session store (for sessions from imported .promptplay files -- use `originalFilePath`)
4. Add `storedSessions` and `loadStoredSessions` from `useSessionStore` to the destructured values at the top of Builder.
5. Call `loadStoredSessions()` in the existing `useEffect` on mount (add it alongside `discover()` and `loadPresentations()`).
6. Create handler: `handleOutlineSessionClick(sessionId: string)` that:
   - Looks up filePath from discoveredSessions first: `discoveredSessions.find(s => s.sessionId === sessionId)?.filePath`
   - Falls back to storedSessions: `storedSessions.find(s => s.sessionId === sessionId)?.originalFilePath`
   - If found, calls `parseSession(filePath, sessionId)`
7. Pass `onSessionClick={handleOutlineSessionClick}` to `<PresentationOutline>`

**Styling for clickable session entries:**
- Add a subtle hover effect (background color change) to signal clickability
- The cursor should be `pointer` on hover
