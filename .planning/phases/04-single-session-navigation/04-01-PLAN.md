  <action>
    Create `src/renderer/src/stores/navigationStore.ts` following the existing Zustand patterns in sessionStore.ts (same `create` import, same state+actions pattern).

    **State shape:**
    ```typescript
    interface NavigationState {
      steps: NavigationStep[]
      currentStepIndex: number
      expandedSteps: Record<number, { user: boolean; assistant: boolean }>

      initializeSteps: (messages: ParsedMessage[]) => void
      nextStep: () => void
      prevStep: () => void
      goToStep: (index: number) => void
      goToFirst: () => void
      goToLast: () => void
      toggleExpand: (stepIndex: number, role: 'user' | 'assistant') => void
      reset: () => void
    }
    ```

    **Action implementations:**
    - `initializeSteps(messages)`: Call `filterVisibleMessages(messages, false)` then `buildNavigationSteps(result)`. Import both from `../utils/messageFiltering`. Set steps, reset currentStepIndex to 0, clear expandedSteps to `{}`. Per user decision: responses start collapsed (expandedSteps empty = all collapsed).
    - `nextStep()`: If `currentStepIndex < steps.length - 1`, increment currentStepIndex.
    - `prevStep()`: If `currentStepIndex > 0`, decrement currentStepIndex.
    - `goToStep(index)`: Bounds-check (0 <= index < steps.length), set currentStepIndex.
    - `goToFirst()`: Set currentStepIndex to 0.
    - `goToLast()`: Set currentStepIndex to `steps.length - 1` (guard against empty steps array).
    - `toggleExpand(stepIndex, role)`: Read current expandedSteps, get entry for stepIndex (default `{ user: false, assistant: false }`), toggle the specified role's boolean. Write back with spread to create new object reference. Per user decision: expand/collapse state is remembered across navigation (stored in Zustand, not component state).
    - `reset()`: Set steps to [], currentStepIndex to 0, expandedSteps to {}.

    **Important:** The store does NOT import from any component. It only imports from utils/messageFiltering and types/pipeline. This keeps the dependency graph clean.

    Export as `useNavigationStore`.
  </action>
